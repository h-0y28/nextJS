# Library vs Framework

react = libary
next.js = framework

### Library

- 코드 내에서 사용하는 것
- 원하는 아키텍처를 사용하여 워나는 방식으로 코드를 작성함
- 코드 내에서 사용하려고 설피하는 것이지만 **사용의 주체는 개발자임**
- 다운 받아서 사용하는 것

### Framework

- framework가 개발자의 코드를 사용함
- **개발자의 주도권 X**
- **제시된 규칙을 지켜야 하는 것**
- 올바른 위치에 코드를 넣으면 framework가 그 코드를 가져와서 사용 (next.js같은 경우 풀스택 웹 애플리케이션을 빌드 해줌)
